# Easy Tool Kit

<img src="./resources/logo.ico" width="200" /> 

Kit de ferramentas para gerenciar o download, a descriptografia e processo de indexação de evidências.

### Siga as instruções para baixar, instalar e utilizar.

1 - [Clique aqui para baixar a versão v0.1.7](https://github.com/izidorio/easy-tool-kit/releases/download/v0.1.7/easy-tool-kit-v0.1.7.zip)

2 - Descompacte o arquivo .zip baixado na pasta de sua preferência. 

3 - Clique 2 vezes no executável `easy-tool-kit.exe`

4 - Clique em Settings e selecione o arquivo `git-bash.exe` e o arquivo `iped.exe`

> Caso você não tenha o Git instalado na sua máquina siga as instruções:
> Abra o Powershell e execute o comando abaixo para instalar o Git Bash: `winget install --id Git.Git -e --source winget`

> Você vai precisar do IPED. Caso você não tenha o IPED instalado siga o [tutorial da AFD clicando aqui](https://academiadeforensedigital.com.br/sistema-iped-forense/)
